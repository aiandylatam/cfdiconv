# CFDIconv

Convierte tu ZIP del SAT en algo que sí entiendes.

App 100% client-side para analizar, visualizar y exportar CFDIs (XML del SAT mexicano). Todo corre en tu navegador — ningún archivo sale de tu máquina.

## Features

- Drag & drop de XMLs sueltos o ZIP completo del SAT
- Soporte CFDI 3.3 y 4.0 (Ingreso, Egreso, Pago, Nómina, Traslado)
- Auto-detección de tu RFC
- Dashboard con KPIs, gráficas mensuales, top clientes/proveedores
- Resumen fiscal (IVA trasladado/acreditable, retenciones)
- Detección de UUIDs duplicados
- Exporta a PDF (3 plantillas), Excel multi-hoja, CSV, resumen ejecutivo

## Stack

Vanilla JS, sin build. Librerías por CDN: jsPDF, JSZip, SheetJS, Chart.js, QRCode.

## Uso

Abre `index.html` en cualquier navegador moderno.
